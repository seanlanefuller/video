# Selfie App

A simple, lightweight web application for taking selfies using your webcam.

## Features

- **Webcam Access**: Requests permission to use your camera directly in the browser.
- **Snapshot Capture**: Capture a high-quality snapshot from the live video feed.
- **Auto-Download**: Automatically downloads your selfie as a PNG file.
- **Dark Mode UI**: Clean, minimalist dark interface for better focus.

## Installation

No installation is required. This is a standalone HTML project.

1. Clone the repository:
   ```bash
   git clone https://github.com/seanlanefuller/video.git
   ```
2. Open `index.html` in any modern web browser.

## Tech Stack

- **HTML5**: For structure and layout.
- **CSS3**: For styling and responsive design.
- **JavaScript**: For camera integration and image processing.
